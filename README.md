# Sales Dashboard with Streamlit

Interactive sales dashboard built with Python and Streamlit using sample business data.

This project demonstrates how raw sales data from CSV files can be transformed into a simple and useful dashboard for business analysis, KPI tracking and decision-making.

## Overview

Many small businesses store sales data in spreadsheets but do not have a clear way to monitor performance, identify trends or understand their key indicators.

This dashboard provides a practical example of how to turn spreadsheet data into visual insights using Python.

## Features

- Import sales data from CSV
- Display key business indicators
- Filter sales by date, product category and sales channel
- Analyze revenue trends
- Identify revenue by product category
- Compare sales performance by channel
- Display filtered source data

## Example KPIs

- Total revenue
- Number of orders
- Average ticket
- Total units sold
- Revenue by category
- Revenue by sales channel
- Monthly sales trend

## Tech Stack

- Python
- Streamlit
- Pandas
- Plotly
- CSV
